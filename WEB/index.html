Información sobre la página.

Bienvenidos a nuestra aplicación web, diseñada para ofrecer una experiencia segura y accesible desde cualquier dispositivo móvil. Esta aplicación permite a los usuarios, utilizando las coordenadas de su ubicación actual o introduciendo manualmente coordenadas GPS, visualizar información detallada sobre los embalses que se encuentran en un radio ajustable de hasta 1000km. Los usuarios pueden personalizar este radio según sus necesidades.

Entre las funcionalidades principales se encuentran:

  • Visualización de embalses cercanos: Se muestran los embalses dentro del radio de acción configurado por el usuario, con información extraída de nuestras bases de datos.
  • Filtrado avanzado: Los usuarios pueden realizar filtrados por cualquiera de los atributos de los embalses, como la demarcación, cantidad de agua almacenada, o uso del embalse.
  • Estadísticas personalizadas: La aplicación permite calcular y visualizar estadísticas importantes, como máximos, mínimos, medias y otros valores clave. Esto incluye datos específicos de la cantidad de agua embalsada al final del año hidrológico o natural, por embalse, ámbito o demarcación.

Además, para garantizar un uso eficiente y seguro de la aplicación, hemos implementado medidas de seguridad avanzadas que previenen ataques como el web scraping o la denegación de servicio (DoS). Estas medidas aseguran que la aplicación funcione de manera óptima incluso frente a consultas masivas provenientes de herramientas automatizadas.

Disfruta de una experiencia completa, segura y funcional con nuestra aplicación para la consulta y análisis de datos de embalses.
